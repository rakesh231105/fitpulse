## 💚 Project Overview

FitPulse is a health monitoring and anomaly detection system designed to analyze fitness data such as heart rate, steps, and oxygen saturation (SpO2). The system identifies unusual patterns in user health metrics and provides actionable insights to improve overall well-being.

This project was developed as part of the **Infosys Springboard Program**, focusing on applying data analysis and machine learning techniques to real-world healthcare scenarios.

The application includes a complete pipeline starting from data preprocessing and feature engineering to anomaly detection and visualization through an interactive dashboard. Users can input their fitness data or upload datasets, and the system processes the data to detect anomalies and classify them based on severity levels.

The dashboard presents key metrics, trends, and meaningful health recommendations such as increasing physical activity, maintaining optimal heart rate, and monitoring oxygen levels. This enables users to make informed health decisions through a simple and user-friendly interface.
